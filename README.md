# 🚀 AttendX — Personal Attendance Tracker

AttendX is a modern web-based attendance tracking system built using Flask.  
It helps students track subject-wise attendance, calculate overall percentage, and predict future attendance to maintain the required 75% criteria.

🔗 **Live Demo:**  
https://attendx-9b5k.onrender.com

---

## ✨ Features

- ➕ Add new subjects
- ✏ Update attendance records
- ❌ Delete subjects
- 📊 Automatic attendance percentage calculation
- 📈 Future attendance prediction
- ⚠ Warning for attendance below 75%
- 💾 SQLite database integration
- 🌐 Fully deployed on Render
- 🎨 Clean and modern UI

## ⚙ Installation (Run Locally)

1️⃣ Clone the repository:


git clone https://github.com/sadhnasorouthr-create/attendx.git

cd attendx


2️⃣ Create virtual environment:


python -m venv venv
venv\Scripts\activate


3️⃣ Install dependencies:


pip install -r requirements.txt


4️⃣ Run the application:


python app.py


5️⃣ Open in browser:


http://127.0.0.1:5000
